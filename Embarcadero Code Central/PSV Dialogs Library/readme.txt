INSTALLATION:

1. Delphi 5.x:

Uninstall previous installed version of psvDialogs Library from Delphi 5 IDE.
Remove previously compiled psvDialogs package psvdr5.bpl from your hard disk.

Use "File\Open..." menu item of Delphi IDE to open psvDialogs runtime
package PSVDR5.DPK. In "Package..." window click "Compile" button to
compile packages PSVDR5.DPK. 
Put compiled BPL file into directory that is accessible through the search PATH (i.e. DOS
"PATH" environment variable; 
for example, in the Windows\System directory).

After compiling psvDialogs run-time package you must install design-time
package into the IDE.

Use "File\Open..." menu item to open design-time package PSVDD5.DPK.
In "Package..." window click "Compile" button to compile the package
and then click "Install" button to register psvDialogs Library 
components on the component palette.
