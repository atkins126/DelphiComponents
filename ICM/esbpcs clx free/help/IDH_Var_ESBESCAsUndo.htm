ESBESCAsUndo Global Variable
When set to true then ESC is treated as though it were Ctrl-Z (Undo) when pressed in an ESBEdit Field.

Unit
QESBPCSGlobals

Declaration
ESBESCAsUndo : Boolean = True;

Description
Set to False if you don't want this behaviour.
