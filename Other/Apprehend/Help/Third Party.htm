ImagePainter

ImagePainter is a Image editor component that comes with over 20 built in ready to use tools that provides basic GDI functions like line, ellipse and so on.

Image painter may be downloaded here.

http://www.omuller.com/

An Apprehend\ImagePainter demo is provided. Download and install ImagePainter, the open and run the ApprehendImagePainterDemo.
