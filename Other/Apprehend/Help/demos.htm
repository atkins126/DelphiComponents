Demos for  mbarcadero Technologies TM Delphi TImage component, TImageENView and TImageENVect by are provided to show how easy it is to use Apprehend with most image libraries.

Demos for the ImageEn, Image Libraries provide additional features and file support not possible with TImage. Additional file support may be added to TImage by employing use of GraphicsEx, ImageENIO or Envision Image library.

None of the components used in the demos are provided.  ImageEN must be obtained from here:
Those who wish to test the functionality of Apprehend with Delphi 5 can install the Apprehend component, and then recompile the TImage Demo.  No additional components are required for the TImage demo other than those provided with Delphi 5-7.  Apprehend is not available for Delphi 8 and Delphi 2005.  Packages are not provided for Delphi XE or XE2.
 

Demo Projects
