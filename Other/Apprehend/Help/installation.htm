Packages are provided for Delphi 5, 6, 7 and Delphi 2010.

Delphi
  • Delphi 5: ASG_Capture_D5.dpk
  • Delphi 6: ASG_Capture_D6.dpk
  • Delphi 7: ASG_Capture_D7.dpk
  • Delphi 2010: ASG_Capture_D13.dpk

Delphi XE and XE2: 

  • Create a new package.  
  • Add all *.pas files to the package,
  • Compile and Install.
  • Add Path to the source folder to the IDE LibaryPaths.
