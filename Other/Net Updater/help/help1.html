NetUpdater Introduction

Are your customer's using the LATEST version of your software?

Now keep ALL your customer's applications up-to-date. Software development is an evolutionary process. As soon as a new application is launched, developers start creating updates. Managing the update process can be overwhelming at the best of times. Net Update provides a flexible, easy solution to manage all your customer's update requirements.

NetUpdater (TkmNetUpdate) is the fastest and easiest way to distribute all your software updates. Using a powerful client/server infrastructure, client software can be modified, patched, or completely updated in whatever configuration best suits your requirements. Flexible, secure, and reliable. NetUpdater is a simple, yet comprehensive software update solution built on trusted, industry-safe Internet and LAN protocols. Optionally, end users can be directed to a URL where more information about the update is available, and files can be downloaded from there.


Key benefits of using NetUpdater include:

     Cost Efficiency
     Flexible Architecture
     Industry Standard Protocols
     Easy Integration
     Royalty Free Distribution



Cost Efficiency
Managing software updates is a costly process. The time required by software developers to keep their distrubted applications current can be considerable. This is time that developers would rather spend improving and marketing their product, rather than distributing updates.

Flexible Architecture
Designed from the ground up to be flexible, yet feature rich. The easy-to-use NetUpdater editor provides all options to manage update information on your update server in a comfortable Wizard driven interface. Depending on the type of update to be deployed, NetUpdater can replace the require files itself, run an external setup file, or simple redirect the user to a URL for more information.

Industry Standard Protocols
NetUpdater is built on the same trusted, dependable industry standards you currently rely on. All Internet communication is controlled by Microsoft's WinInet API used in today's standard Windows and Internet Explorer applications. NetUpdater supports all of the most common client/server communication protocols, including HTTP, HTTPS, FTP and LAN (both UNC paths and mapped drives). Full support for proxy authentication, HTTP and FTP user authentication are built right in.

Easy Integration
NetUpdater's visual development environment was designed to create advanced update systems without using a single line of code. Simply drop the component on your form, set a few properties and you have created a powerful application update system. Now use the provided editor to configure your server information, and you are ready to go. NetUpdater offers you a powerful, yet easy way to deploy your updates to your customers. The advanced customization capabilities of NetUpdater give you complete control of all the options available.

Royalty Free Distribution
The registered verions of NetUpdater allows the developer to freely support an unlimited number of applications created by the developer.

Easily integrate any product update into your application or business processes. There is no other software of it's kind that is more flexible or as easy to use.

NetUpdater is the ONLY sensible choice!
