Component 
Version: 	1.5.0.0

Component 
Description: 	TDBDateTimePicker is a data-aware descendant of native TDateTimepicker control.
		This TDBDateTimePicker control also displays a blank value for a no date value.

New:
	Properties -
		DataField: String - Specifies the field which the DBDateTimePicker control 
			   displays data.
		DataSource: TDataSource - Links the DBDateTimePickerControl to the dataset
			    that contains the field it represents.
