Installation:

This example shows the installation of components by Delphi 6. 

   1. Unpacking the archive (CHPack_free.zip) into a folder.
   2. entering the folder "..\CHPack\D6" in the search path of the component library.
   3. choosing the file "CHComponnetsD6.dpk" in Delphi about File|Open 
      (according your Delphiversion).
   4. press the button "Package install" and the Package will be installed.
   5. Ready
