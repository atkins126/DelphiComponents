Описание библиотеки

SoftMastеr VCL Extensions Library (SMVCL) for Delphi - это библиотека компонентов и функций для Borland Delphi 6-7 и Builder C++. Данная библиотека является бесплатным свободно распространяемым (FreeWare) продуктом. Вы вольны свободно распространять библиотеку в некоммерческих целях в оригинальном виде, без изменения исходных текстов модулей и содержимого инсталляционного архива. Для коммерческого использования данного продукта, вы должны зарегистрировать продукт. Зарегистрированным пользователям предоставляются исходные тексты. Уточнить процесс регистрации можно по адресу указанный ниже.

Правильность работы компонент не гарантируется. Ответственность за возможный ущерб в результате использования данного программного кода полностью перекладывается на Вас. Однако автор принимает любые замечания о неверной работе компонентов с целью скорейшего устранения обнаруженных Вами ошибок, а также любых пожеланий.

 

Установка
-----------
Для установки необходимо установить набор компонент SoftMaster Library в палитру компонент Delphi. Для этого:

==== Delphi 6 и 7
Откройте файл SoftMaster.dpk
В появившемся окне "Package..." нажмите кнопку [Install].

После перекомпилляции библиотеки в палитре компонент Delphi
появиться новая страничка

Добавление путей
- в Delphi IDE, выберите "Tools|Environmet options..."
- переключитесь на закладку "Library", строка ввода "Library path"
- добавьте путь к папке "SoftMaster"

 

Краткое описание

Визуальный компоненты:

TDBTreeView - является наследником компонента TTreeView. Компонент DBTreeView предназначен для отображения и управлением древовидной иерархической структурой в базе данных. Он отображает информацию в виде некой виртуальной файловой системе: в виде папок и объектов.

TSmIPEdit - компонент предназначенный для ввода и редактирования IP адреса

 

Невизуальные компоненты:

TSmLogEvent - простой компонент для ведения журнала различных событий. В журнал записывается дата, время и описание события, определенное пользователем.

TSmWinList - компонент предназначен для получения списка открытых окон.

TSmExportToExcel - производит конвертацию данных из определенного TDataSet в файл Excel. Есть возможность просмотра хода выполнения преобразования
