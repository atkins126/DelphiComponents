Component TSmExpotToExcel
Main page

 

Component TSmExpotToExcel converts the data from specified TDataSet in Excel file. There is a possibility to view a progress of proccess.

Property

  • property FileName: string - specifies the name of Excel file, in which converting data
  • property DataSet: TDatSet - specifies the dataset to use with the current TSmExpotToExcel component.
  • property Header: TStrings - list, specifies user's headers
  • property Footer: TStrings - list, specifies user's footer
  • property Captions: TFiledCaptions - type of captions
  • Type TFiledCaptions = (smNone, smLabels, smFieldNames); - smNone - with out header, smLabels - specified by user in property Header, smFieldNames - captions are the same as name of field in DataSet
  • property CaptionBgColor: TColor - color of background of caption

Method

  • method Execute - run conversion.

Event

  • Type TNotifyEvent = procedure(Sender: TObject; Progress: integer) of object;
  • property OnExportProgress: TNotifyEvent - event occurs after conversion each row of data. Value Progress reflect percent execution of conversion.
