Компонент TSmExpotToExcel
Главная страница

 

Компонент TSmExpotToExcel производит конвертацию данных из определенного TDataSet в файл Excel. Есть возможность просмотра хода выполнения преобразования.

Свойства компонента

  • property FileName: string - определяет имя файла Excel, в который конвертируются данные
  • property DataSet: TDatSet - набор конвертируемых данных
  • property Header: TStrings - список, определяющий пользовательские заголовки
  • property Footer: TStrings - список, определяющий пользовательский "подвал"
  • property Captions: TFiledCaptions - тип как выводятся заголовки столбцов
  • Type TFiledCaptions = (smNone, smLabels, smFieldNames); - smNone - без заголовка, smLabels - определенные пользователем в свойстве Header, smFieldNames - заголовки аналогичны названием соответствующих полей
  • property CaptionBgColor: TColor - цвет фона заголовка

Методы компонента

  • method Execute - запуск конвертации.

События компонента

  • Type TNotifyEvent = procedure(Sender: TObject; Progress: integer) of object;
  • property OnExportProgress: TNotifyEvent - событие возникающее при конвертации каждой строки данных. Значение Progress при использовании данного события отражает процент выполненной конвертации
