Component TSmExpotToExcel
Main page

 

Component TSmIPEdit intend for entering and editing of IP address.

Property

  • property IPAddr: string - specifies IP address
  • property Color: TColor - specifies the background color of the control.
  • property Ctl3D: Boolean - determines whether a control has a three-dimensional (3-D) or two-dimensional look.

Event

  • Type TNotifyEvent = procedure(Sender: TObject; Progress: integer) of object;
  • property OnExportProgress: TNotifyEvent -occurs when the IP address for the control may have changed. Varibale Part indicates which part of IP adress is edited in this moment.
