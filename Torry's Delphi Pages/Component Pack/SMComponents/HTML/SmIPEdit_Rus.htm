Компонент TSmIPEdit
Главная страница

 

TSmIPEdit - компонент предназначенный для ввода и редактирования IP адреса.

Свойства компонента

  • property IPAddr: string - определяет IP адрес
  • property Color: TColor - определяем цвет фона компонента
  • property Ctl3D: Boolean - определяет как выглядит компонент: как 3D или 2D

События компонента

  • Type TNotifyEvent = procedure(Sender: TObject; Part: integer) of object;
  • property OnChange: TNotifyEvent - событие возникающее при измении значения IP адреса. Значение Part указывает на одну из четырех частей IP адреса которая меняется в данный момент.
