Component TSmLogEvent
Main page

 

Component TSmLogEvent intend for logging of the different events. In file of log entering date,time and description of the event.

Property

  • property FileName: string - define the file name, in which the events log.

Method

  • method Log(Event: string) - write Event in the file with date and time. If log file is absent, it will be created before writing.
  • method procedure DeleteLog - delete log file..

Event

  • property OnLogEvent: TNotifyEvent - the event occurs after executed writing in log file.
