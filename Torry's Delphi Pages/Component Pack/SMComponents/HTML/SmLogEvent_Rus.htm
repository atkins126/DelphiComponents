Компонент TSmLogEvent
Главная страница

 

Компонент TSmLogEvent предназначен для ведения журнала различных событий. В файл журнала записывается дата, время и описание события, определенное пользователем.

Свойства компонента

  • property FileName: string - определяет имя файла, куда записываются события

Методы компонента

  • method Log(Event: string) - производит запись события Event в файл с указанием даты и времени. Если файл отсутствует, то он при этом создается.
  • method procedure DeleteLog - удаляет файл журнала.

События компонента

  • property OnLogEvent: TNotifyEvent - событие происходит после того как произведена запись в файл журнала.
