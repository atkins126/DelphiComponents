Component TSmWinList
Main page

 

Component TSmLogEvent intend for getting list of open windows, and their hadles. There is possible filtering of list: all windows or just visible. Besides there is possible pick out the name and handle of the active window.

Property

  • property ActiveWinHdl: LongWord - handle of active window
  • property ActiveWinName: string - name of active window
  • property Count: integer - the count of open windows
  • property WinType: TTypeOfWindows - type of getting list: all window or just visible.
  • Type TTypeOfWindows = (wnAll, wnVisible); wnAll - all windows, wnVisible - visible windows.
  • property Items[Index: integer]: TWinList; - Individual window.
  • TWinList = record
    WinName: string; - name of window
    Hdl: HWND; - handle of window
    end;

Methods

  • method Refresh; - refresh the list of opening windows.

Example creation of component in run-time, getting the list of visible windows and extraction of the active window.

      var
          SmWinList1: TSmWinList;
          i: integer; 
      begin
          SmWinList1 := TSmWinList.Create(nil);
          SmWinList1.WinType := wnVisible;      //Defining that we need only visible windows
          SmWinList1.Refresh;         //Getting the list of windows
          for i:=0 to SmWinList1.Count-1 do        //Processing the list open windows
              begin
              StringGrid1.Cells[0,i+1] := SmWinList1[i].WinName;
              StringGrid1.Cells[1,i+1] := IntToStr(SmWinList1[i].Hdl);
             end;

         StringGrid1.RowCount := SmWinList1.Count+1;
         Edit1.Text := IntToStr(SmWinList1.Count);        //Count of windows
         Edit2.Text := SmWinList1.ActiveWinName;       //The name of active window
         Edit3.Text := IntToStr(SmWinList1.ActiveWinHdl);      //Handle of active window
     end;
