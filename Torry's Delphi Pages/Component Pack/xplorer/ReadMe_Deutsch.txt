XPLorer 1.1

Das XPLorer-Komponentenpackage beinhaltet fünf Komponenten, mit denen Delphi-Programme ans Design von Windows XP angepasst werden können:

- TXPInformationPanel und TXPInformationBar ermöglichen es, die Detailpanels des Windows Explorer nachzubilden, können aber auch einzeln verwendet werden, um Ordung in selbstentwickelte Programme zu bringen.
- TXPStyler passt das Design von Delphi-Komponenten auf Systemen mit Windows XP an das gewählte Windows-Theme an.
- TXPInformationDialog erzeugt einen Info-Dialog, der dem von Windows XP-Programmen gleicht.
- TXPBubbleHint ist geeignet, um Informationen in Sprechblasen anzuzeigen. Anders als die meisten vergleichbaren Komponenten funktioniert diese auch unter Windows 95 und Windows 98.
