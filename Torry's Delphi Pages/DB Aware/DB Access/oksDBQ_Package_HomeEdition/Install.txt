oksDBQueryBuilder Home Edition V1.2




Installation
------------

on all Delphi 32 bit version since Delphi 7 to 2005
note: oksDBEngine installation is required




1. Copy the zip-file "Package_HomeEdition.zip" to any desired directory
2. Extract all files from zip-file to this directory (all files must be in same directory!)
3. Run Delphi 
4. In all Delphi versions select file -> open -> oksDBQueryBuilder_HomeEdition.dpk
5. Compile und Install the package
6. Save and close the package
7. The component TOKSDBQueryBuilder HomeEdition should have been installed to the components tab "oksDBEngine"
