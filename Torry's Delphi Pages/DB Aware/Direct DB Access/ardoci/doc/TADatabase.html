TDatabase

Свойства
property Active:boolean; Открывает/закрывает database.
Методы
constructor Create(AOwner:TComponent); override; Создает обьект TADatabase. Инициализирует внутренние переменные.
destructor Destroy; override; Разрушает database и закрывает (Close) все DataSetы привязаные к нему.
procedure Open;abstract; Метод для установления соединения с базой.
procedure Close;abstract; Разрыв соединения с базой. Все DataSets привязаные к этому database будут закрыты.
procedure AddDataSet(DataSet:TDataSet); Добавляет DataSet в список прилинкованых DataSetов. (для внутреннего использования)
procedure RemoveDataSet(DataSet:TDataSet); Удаляет DataSet из списка прилинкованых DataSetов. (для внутреннего использования)
